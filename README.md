## ✅ How to Use This Repo

1. Clone the repo:

```bash
git clone https://github.com/rd1855/devops-journey.git
cd devops-journey
```

2. Navigate to the desired day folder:

```bash
cd Day02
```

3. Run scripts (if applicable):

```bash
chmod +x script_name.sh
./script_name.sh
```
